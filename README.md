# Emanuel's  dotfiles
My Hyprland dotfiles to my own OS experience
## Default Components -
### Essentials
* Authenticaion agent - hyprpolkitagent
* Terminal - kitty
* Notification deamon - SwayNotificationCenter
* Application launcher - rofi
* File manager - nautilus
* Pipewire - pipewire
* XDG Desktop Portal - xdg-desktop-portal-hyprland
* Shell - fish 
* Clipboard - wl-copy
### Extras
* hyprshot
* waybar
* hyprpaper
* hypridle
* hyprlock
* wlogout
## Apps
### Essentials
* Browser - Brave
* Video Player - Gnome Video Player
### Extras

## Shortcut / Keybinds -
| Keybind | Action |
|---|---|
| SUPER + T | Open Terminal |
| SUPER + B  | Open Browser |
| SUPER + F  | Open Rofi / Spotlight |
| SUPER + Q  | Quit app |
| SUPER + E | Open Files / Nautilus |
| ALT + Shift + S | Open Spotify |
| SUPER + Shift + O | Open Settings |
